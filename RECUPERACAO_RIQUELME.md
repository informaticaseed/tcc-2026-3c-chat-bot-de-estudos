# Respostas

## 1. O que é uma Issue e para que serve no desenvolvimento de software?

Uma Issue é um registro usado para organizar tarefas, problemas ou melhorias em um projeto. Ela ajuda a equipe a acompanhar o que precisa ser feito, quem é o responsável e o andamento de cada atividade.

## 2. O que é um Pull Request e por que ele é importante?

Um Pull Request é um pedido para juntar as alterações feitas em uma branch ao projeto principal. Ele é importante porque permite que outras pessoas revisem o código, encontrem erros e garantam que tudo esteja funcionando antes da alteração ser aceita.

## 3. O que é separação de camadas no código?

A separação de camadas organiza o código para que cada arquivo tenha uma função específica. O `app.py` normalmente controla a execução do programa e a interação com o usuário, enquanto o `repositorio.py` é responsável por acessar e manipular os dados. Isso deixa o código mais organizado, fácil de entender e de manter.
